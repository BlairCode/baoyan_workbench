# 推免准备本地工作台

> 一个本地优先、隐私友好的推免/保研准备管理系统。  
> 用一个轻量网页集中管理院校项目、导师套磁、材料文件、待办事项和面试题库。

## 为什么做这个

推免准备经常会同时出现这些东西：

- 院校项目、夏令营、预推免、九推进度混在一起。
- 导师套磁信、近期论文、主页信息、回复状态散落在不同文件夹。
- 简历、成绩单、证明、报名材料、面试稿来回改版本。
- 表格能记录进度，但打开本地文件、关联导师和院校并不顺手。

这个项目的目标很简单：  
把“表格记录”和“本地资料”接到同一个网页里，方便读、写、查、打开和备份，同时不把个人材料上传到任何云端。

## 核心功能

| 模块 | 能做什么 |
| --- | --- |
| 总览 | 查看夏令营报名数、套磁进度、待办数量和关键入口 |
| 套磁 | 按导师管理状态、套磁信、相关论文和本地文件 |
| 资源 | 按一级/二级文件夹浏览材料，支持搜索、上传、归类、备注、预览和删除 |
| 院校 | 管理学校、学院/项目、阶段、时间、账号、状态和备注 |
| 待办 | 跟踪任务范围、截止日期、优先级、状态和备注 |
| 面试 | 整理面试题、答案要点、主题和标签 |
| 设置 | 修改头像、标题、工作台名、首页横幅和整体配色 |
| 备份 | 一键备份本地 SQLite 数据库 |

## 适合谁

- 正在准备推免、保研、夏令营、预推免、九推的同学。
- 需要同时管理多个学校、多个导师和大量申请材料。
- 希望本地保存简历、成绩单、证明、套磁信、导师论文等敏感文件。
- 想要一个比纯 Excel 更顺手、比在线系统更私密的个人工作台。

## 特点

- 本地优先：默认运行在 `127.0.0.1`，数据保存在本机。
- 隐私友好：个人材料、数据库和备份目录默认被 `.gitignore` 忽略。
- 轻量直接：只依赖 Python 标准库、SQLite、原生 HTML/CSS/JavaScript。
- 文件友好：网页里记录信息，本地文件仍用系统默认程序打开。
- 可改造：没有复杂框架，适合按自己的推免流程继续改。

## 快速开始

确保已安装 Python 3。

```powershell
python app.py
```

浏览器打开：

```text
http://127.0.0.1:8848
```

Windows 用户也可以双击：

```text
start.bat
```

如需更换端口：

```powershell
$env:BAOYAN_PORT="8891"
python app.py
```

## 推荐目录结构

公开仓库建议只保留源码、README 和已脱敏截图：

```text
.
├─ app.py
├─ start.bat
├─ README.md
├─ .gitignore
├─ web/
│  ├─ index.html
│  ├─ app.js
│  └─ style.css
└─ imgs/
   ├─ 总览页.png
   ├─ 套磁页.png
   ├─ 资源页.png
   ├─ 院校页.png
   ├─ 待办页.png
   └─ 面试页.png
```

本地运行后会自动使用或生成私有目录：

```text
data/
├─ app.db
└─ backups/

保研准备/
└─ 网页添加/
```

建议把自己的材料放在 `保研准备/` 下，例如：

```text
保研准备/
├─ 基本材料/
├─ 套磁信/
├─ 论文/
├─ 院校项目/
├─ 夏令营/
├─ 面试/
└─ 网页添加/
```

网页资源页会扫描本地文件，并按一级/二级文件夹展示。

## 使用建议

1. 将个人资料放在 `保研准备/` 下。
2. 在网页左侧点击“扫描材料”，同步本地文件索引。
3. 在资源页为文件设置归类：`基本材料`、`套磁`、`院校`、`项目`、`面试`、`参考`。
4. 归类为 `套磁` 时，可以选择关联导师。
5. 归类为 `院校` 时，可以选择关联院校。
6. 在套磁页使用“套磁信”和“相关文件”快速打开导师相关材料。
7. 在院校页使用“相关文件”快速查看某个院校项目的材料。
8. 在左侧“设置”中修改头像、标题、首页横幅和整体风格。
9. 经常点击“备份数据”，备份 SQLite 数据库。
